# Reports

- Week0_Final_Report.md: Medium-style writeup to export as PDF.

How to export to PDF:
- Option A (VS Code): Open the Markdown, right-click, "Open Preview" and print to PDF.
- Option B (Pandoc): `pandoc reports/Week0_Final_Report.md -o reports/Week0_Final_Report.pdf`
- Option C (Jupyter): Paste content into a notebook Markdown cell and use File > Export As > PDF.
